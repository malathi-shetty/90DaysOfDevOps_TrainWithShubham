Linux follows a layered architecture:

User → Application → Shell → Kernel → Hardware

---

## 1. Applications
Applications are programs used by users:
- VS Code
- Browser
- Terminal

---

## 2. Shell
- Interface between user and kernel
- Executes commands
- Example commands: ls, cd, mkdir

---

## 3. Kernel
- Core of Linux
- Manages CPU, memory, devices
- Communicates with hardware

---

## 4. Hardware
- CPU, RAM, Disk
- Input/Output devices

---

## Command Flow
Command → Shell → Program → Kernel → Hardware → Output

---

## Key Points
- Everything is a file
- Every running task is a process
- Shell acts as middleman
- Kernel is the core of OS
